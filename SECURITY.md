# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in GWS, please report it privately via email to the repository owner rather than opening a public issue.

Include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact

You should receive an acknowledgment within 48 hours. We will work with you to understand the issue and coordinate a fix before any public disclosure.

## Scope

GWS handles worker authentication tokens, governance policy enforcement, and controls execution scope for automated workers. Vulnerabilities in any of these areas are especially relevant.
